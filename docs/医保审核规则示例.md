# 医保审核规则示例

本文档作为医保智能审核系统的测试知识库文档。

## 一、药品使用规范

### 1.1 抗生素使用规定

**违规情形：**
- 无明确细菌感染指征使用抗生素
- 超出医保目录限定支付范围使用抗生素
- 高档抗生素作为首选药物使用

**合理用药标准：**
- 必须有细菌学检查或明确的感染指征
- 优先使用基本药物目录内的抗生素
- 三代及以上头孢菌素需经专家会诊

### 1.2 辅助用药限制

**限制品种：**
- 部分营养类药物
- 部分中成药注射剂
- 疗效不明确的辅助治疗药物

**使用标准：**
- 需符合药品说明书适应症
- 不得作为常规预防性用药

## 二、检查项目规范

### 2.1 CT/MRI检查规定

**违规情形：**
- 无临床指征进行全身CT检查
- 重复进行相同部位的CT检查（间隔小于3个月）
- 门诊患者进行PET-CT检查

**合理使用标准：**
- 必须有明确的临床检查指征
- 普通X光或B超可以诊断的不做CT
- 复查需间隔合理时间

### 2.2 化验检查规定

**违规情形：**
- 过度检查（大包围式检查）
- 重复检查相同项目
- 检查项目与诊断无关

## 三、收费规范

### 3.1 重复收费

**常见违规：**
- 同一项目多次收费
- 打包项目内单项重复收费
- 一次性耗材多次收费

### 3.2 超标准收费

**违规情形：**
- 超过医保定价标准收费
- 将低价项目按高价项目收费
- 自立项目收费

### 3.3 分解收费

**违规情形：**
- 将打包项目分解为多个单项收费
- 将连续性治疗项目按次分解收费

## 四、不同医保类型报销规定

### 4.1 职工医保

- 门诊起付线：按当地规定
- 住院报销比例：在职85%，退休90%
- 乙类药品个人先自付10%

### 4.2 居民医保

- 门诊统筹按比例报销
- 住院报销比例：一级医院85%，二级75%，三级65%
- 乙类药品个人先自付15%

## 五、常见违规案例分析

### 案例1：超范围用药
患者因感冒就诊，医生开具高档抗生素+丙种球蛋白。
**问题：** 无明确指征使用高档抗生素和血液制品。

### 案例2：无指征检查
患者因上呼吸道感染，医生开具胸部CT+腹部CT。
**问题：** 普通呼吸道感染无需CT检查，过度检查。

### 案例3：超标准收费
某医院将普通材料的手术按照进口材料标准收费。
**问题：** 实际使用与收费不符，涉嫌欺诈骗保。

## 六、审核建议

1. 建立完善的临床路径管理
2. 加强医师培训，提高合理用药意识
3. 完善信息系统自动审核功能
4. 定期开展医保政策培训
5. 建立违规预警机制
